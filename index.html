Small Video Game - University Project

This University Project was setup in order for us to get used to using C# and Unity. Thankfully I was already familiar with Unity. Our game was to take place in a museum in order to promote learning historical facts and at the same time try to make it entertaining. So the Hub had two purposes, on the one hand show to the player what the game is about and offer some information about some of the most famous pirates.

The whole game worked around a Hub. Multiple scaled down objects were placed in order to act as representations of the different levels that the player would have to go through. A box collider is present in front of each in order to teleport the player to said level.

Personal Project - Skip (Abandoned ?)

Skip was meant to be a first person platforming game made using Unity. The platforming was meant to be intimately linked with the music played in the background. The gameplay was meant to fast and exciting. Alongside the simple aspect of first person platforming I wanted to add a speedrunning aspect with a live timer. I could go back to this project in the future but it is not an immediate focus of mine.

Personal Project - Last Contact

Horror game in which you play as a police officer who's been asked to check out the latest lead in a string of disappearances. The latest lead brings you to a house in the suburbs, close family and friends have lost contact with them for the past month now, your job is to understand what happened to them.

All the models were made using blender, the only model that isn't mine is the human character, I use it as a scaling reference. All of the rendered images are made in Blender so this is not what the final project will look like as I'm going to use Unity. However I'm tempted to use Unreal Engine.
